# Django 시작 전에...

- Coding Convention (Python)

  - 한 줄의 문자열은 79자
  - DocString은 72자
  - snake_case 사용
  - 모듈 레벨 상수는 모두 대문자
  - ClassName은 Capitalized Word
  - 한 줄로 된, if, try...except, for, while 구문은 사용하지 않는다.

  

- Django Convention
  - \+ coding convetion 
  - 한 줄의 문자열은 119자 추천
  - DocString은 72자



- Version
  - python==3.8
  - django==3.2
